# Age Validation Application

This web application, developed with HTML, CSS, and JavaScript, allows users to validate their age. Users input their name and age, and the application displays a message indicating whether they are of legal age or not.

## Features

-   User-friendly interface.
-   Input fields for name and age.
-   Dynamic messages based on the entered age.
-   Responsive design.

## Technologies Used

-   HTML
-   CSS
-   JavaScript

## How to Run

1.  Clone this repository:
    ```bash
    git clone https://github.com/Sebs2099/APLI-IMC.git
    ```
2.  Navigate to the project directory:
    ```bash
    cd APLI-IMC
    ```
3.  Open the `Index.html` file in your web browser.

## Project Structure

-   `Index.html`: The main HTML file.
-   `src/css/styles.css`: Stylesheet for the application.
-   `src/js/main.js`: JavaScript code for age validation logic.
-   `LICENSE`: Project license information.
-   `README.md`: This file.
